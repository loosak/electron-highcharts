## Quick start

* `npm init`
* Install Electron

```sh
# Install as a development dependency
npm install electron --save-dev

# Install the `electron` command globally in your $PATH
npm install electron -g
```
* `npm i jquery --save`
* `npm i highcharts --save`

* `electron .`
